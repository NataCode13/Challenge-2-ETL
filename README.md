# Challenge-2-ETL

## Objetivo
Este proyecto analiza la evasión de clientes (churn) en Telecom X a partir de un archivo JSON.
El objetivo es limpiar los datos, explorar patrones de comportamiento y detectar variables asociadas a la evasión del servicio.

## Etapas del análisis
1. Extracción de datos desde JSON.
2. Aplanamiento del JSON con `json_normalize`.
3. Limpieza y transformación de variables.
4. Creación de la variable `Cuentas_Diarias`.
5. Conversión de variables binarias a formato 0/1.
6. Análisis descriptivo y visualización de churn.
7. Conclusiones y recomendaciones.
